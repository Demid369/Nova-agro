## 2. Зелёная энергия: биогаз + солнечные + теплицы

**Query:** `биогаз навоз теплицы солнечные энергия`

**Стартовые узлы:** `Теплицы 100 га`, `Солнечные панели 50 МВт·ч`

**Path (2 hops):**

```
Биогазовая установка 20 МВт·ч
  <--shares_data_with [EXTRACTED]--
Сводка проекта «МОЯ МЕЧТА»
  --conceptually_related_to [EXTRACTED]-->
Теплицы 100 га: 18 400 т овощей, 376 млн цветов
```

**Связи из сводки (EXTRACTED):**

- `Солнечные панели 50 МВт·ч` ↔ `Биогазовая установка 20 МВт·ч` (semantically_similar_to)
- `Биогаз из навоза (метан)` → `Электроэнергия (зелёная)` (community «Зелёная энергия»)
- `Концепт: зелёная энергетика` — отдельный семантический узел

**Вывод:** энергокластер привязан к ядру проекта (C6 «Финансы и KPI» / сводка) и к теплицам. Навоз → биогаз → тепло для теплиц описано в гиперребре `Навоз → биогаз → электроэнергия + тепло → теплицы`.
