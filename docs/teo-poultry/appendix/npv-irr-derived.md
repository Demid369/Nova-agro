# NPV / IRR — производный расчёт (не из xlsx)

> **В исходной финмодели NPV и IRR отсутствуют.** Ниже — восстановление по листу «Экономика» с явными допущениями.

## Источник денежных потоков

| Год | Загрузка | EBITDA, млн ₽ | Примечание |
|-----|----------|---------------|------------|
| 2026 | 0% | 0 | CAPEX **−12 000** млн ₽ |
| 2027 | 35% | 504 | выход на мощность |
| 2028 | 65% | 936 | |
| 2029 | 85% | 1 225 | |
| 2030–2035 | 100% | 1 441 | полная мощность |

- **FCF при 100% загрузке:** 1 426 млн ₽/год (ЧП 226 + амортизация 1 200), лист «Экономика».
- **На рампе:** FCF = EBITDA × (1 − 6% ЕСХН) + амортизация × загрузка (укрупнение; в модели указано «до обслуживания долга»).
- **Долг, проценты, оборотный капитал, ликвидационная стоимость** — **не моделировались** (в xlsx нет).

## Ставка дисконтирования

**10%** — по аналогии с baseline холдинга («IRR при 10%» в `docs/graphify-corpus/00-summary.md`). **Не задана в finmodel xlsx.**

## Результаты

| Горизонт | NPV @ 10% | IRR |
|----------|-----------|-----|
| **10 лет (2026–2035, как в xlsx)** | **−4 289 млн ₽** | **≈1,7%** |
| 20 лет (ещё 10 лет FCF на полной мощности) | ≈+199 млн ₽ | ≈10,3% |

## Интерпретация (честно)

1. **Простая окупаемость 8,3 года** (CAPEX / EBITDA) **не равна** положительному NPV @ 10% на 10-летнем горизонте finmodel.
2. При **10%** и **только 10 годах** из таблицы xlsx проект **убыточен в DCF-смысле**.
3. Для **IRR ≈ 10%** нужен **горизонт ~19 лет** при неизменном FCF после выхода на мощность (без учёта окончания амортизации и без terminal value).
4. Цифры **не для инвестмемо без доработки**: нужны WACC проекта, долг, оборотный капитал, срок жизни активов, сценарии цен.
