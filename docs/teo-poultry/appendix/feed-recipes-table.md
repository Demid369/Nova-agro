## 1. Классификация: что забираем из baseline «кроликов»

### 1.1. Без изменений (общий APK, не «рецепт кролика»)

| Объект | Содержание | Применение для птицы |
|--------|------------|----------------------|
| **ККЗ FRAGOLA** | 262 800 т/год, 2 т/ч | Тот же завод холдинга; птице ~66 712 т/год (~25% мощности) |
| **Tab. 141 — рецепт КРС** | ячмень 28%, овёс 12%, пш/кукуруза 25%, шрот подсолнечника 25%, меласса 5%, MCP 2%, соль 1%, премикс 1%, известняк 1% | **Без изменений** — корм для КРС APK, не для птицы |
| **Tab. 141 — рецепт осетровых** | рыбная мука 41%, кукурузный глютен 10%, дрожжи 5%, пшеница 16%, шрот подсолнечника 20%, рыбий жир 7%, премикс 1% | **Без изменений** — блок УЗВ/осётр, не для птицы |
| **Технология ККЗ** | дробление → смешивание → гранулирование/экструдирование; контроль влажности | **Без изменений** — та же линия FRAGOLA |
| **Сырьевая база холдинга** | шрот подсолнечника и сои с полей + МЖК | **Без изменений** — логика закупки/себестоимости 28 000 ₽/t |
| **Минеральный каркас** | MCP, известняк/мел, соль, премикс | **Без изменений** по типам компонентов; **доли другие** под птицу |
| **Премикс «Старт» для птиц** | витамины A, D3, E, B-группа, Fe, Mn, Cu, Zn, Co, I, Se (corpus §687) | **Без изменений** как класс добавки; **марка/доза** — под группу птицы |

### 1.2. Только кролики (в птицу не переносим)

| Объект | Почему не для пtiцы |
|--------|---------------------|
| **Заголовок Tab. 141 «для кроликов»** | Артефакт docx: **процентов для кроликов в Tab. 141 нет** |
| **Narrative: «15 компонентов, рецептура европейских специалистов»** (`teo/109`, corpus §1691) | Описание логики кролиководства; **таблицы % нет** — не рецепт |
| **40–50% концентратов в зимнем рационе кролика** (`teo/124`) | Физиология кролика (сено, зелёная масса летом); у бройлера/несушки — **100% ПК** |

| **Хвост Tab. 141 после рыбы** | Теплицы/томаты — **мусор docx**, не корма |
