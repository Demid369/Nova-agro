# План экспорта — только дружественные страны РФ

> **Статус:** рабочий план проекта (draft), не firm-контракты.  
> **База finmodel:** сбыт **100% РФ** (консервативно). Экспорт — **операционная цель с 4-го года** полной мощности.  
> **Исключено:** страны, в отношении которых РФ ввела меры в ответ на недружественные действия (ЕС, США, Великобритания, Япония, Южная Корея, Украина и др. — перечень актуализировать на дату отгрузки).

## Логика (честно)

| Факт | Следствие |
|------|-----------|
| Производство **≈18 170 т** мяса/год; **≈82%** выручки — бройлер + яйцо | Экспортируем **только мясо**; яйцо и удобрение — **внутренний рынок** |
| Завод greenfield, Запорожье, участок позже | Первые **2 года** после пуска — **0%** экспорта (VetIS, халяль, логистика) |
| Вся РФ в 2025 ≈ **410 тыс. т** экспорта птицы | Наш **6%** объёма мяса ≈ **1 090 т** — **0,27%** федерального потока; не «перекачка» рынка |
| Экспорт = премия + логистика | **+2%** к опту на экспортной доле; **+11 млн ₽/год** OPEX — **NPV ≈ +40…+60 млн** vs база 0% |

## Целевой профиль (устойчивое состояние, год 4+)

| Параметр | Значение |
|----------|----------|
| Доля экспорта в объёме **мяса** | **6%** (**1 090 т**/год) |
| Доля в **общей выручке** | **≈4,2%** (**235 млн ₽**/год) |
| HS-коды | **0207** (тушки, части), **020714** (бройлер заморож.), **020727** (индейка) |
| Формат | **−18 °C** (85%), **охлаждёнка +2 °C** (15%, EAEU) |
| Халяль | **100%** экспортного потока (отдельный блок УПК) |
| Доп. OPEX экспортной службы | **11 млн ₽/год** (8 FTE, фрахт, ветсопровождение, сертификация) |

### Продуктовый микс экспорта

| Продукт | т/год | млн ₽/год | Доля |
|---------|-------|-----------|------|
| Бройлер (тушка, бедро, крыло, филе) | 925 | 197 | 84% |
| Индейка (тушка, филе) | 110 | 40 | 17%* |
| Утка / перепёл (премиум) | 55 | 18 | 8%* |

## Сертификация и логистика

| Требование | Действие |
|------------|----------|
| VetIS / «Меркурий» | Прослеживаемость с УПК |
| ЕАЭС (KZ, BY, AM, KG) | ТР ТС 021/2011; единый ветсертификат |
| Халяль (MENA, UZ, IR) | Аудит блока УПК; **РДУМ РФ** + при необходимости ESMA/UAE |
| Фито/ветконтроль | Rosselkhoznadzor, соглашения с страной назначения |
| Логистика из Запорожья | Рефконтейнер → порт **Новороссийск / Тамань / Керч** → назначение; бюджет **≈18–22 ₽/кг** море, **≈8–12 ₽/кг** ж/д EAEU |

## Финансовый эффект (расчёт)

| | млн ₽/год |
|--|-----------|
| Выручка экспорт (6% мяса, +2% цена) | **235** |
| Перераспределение из внутреннего рынка | −235 |
| Премия +2% на экспортной доле | **+4,7** |
| Доп. OPEX экспорт | **−11** |
| **Δ EBITDA** | **≈ −6…+5** (около **0** в базе; апсайд при ОАЭ/Иран премии) |
| **Δ NPV @10%, 16 лет** | **≈ +40…+60** (vs 0% экспорт) |

> **Вывод:** экспорт для нас — **диверсификация и халяль-премия**, не второй столб выручки. В finmodel **база 0%**; сценарий «План 6%» — отдельная строка листа «Экспорт».
