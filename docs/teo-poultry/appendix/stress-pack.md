## Критические точки (break-even @10%)

| Драйвер | Порог | NPV ≈ 0 |
|---------|-------|---------|
| Цена **куриного яйца** | падение | **~−28%** при прочих базовых |
| Цена **корма** | рост | **+19…20%** |
| **Ramp** | неполная мощность | **85%** без выхода на 100% → NPV < 0 |
| **Комбинация** | одновременно | проект **не проходит** 10% без мер |
