## Убойно-перерабатывающий комплекс

| Параметр | Значение |
|----------|----------|
| CAPEX УПК | **2 600 млн ₽** |
| Холодильный склад | **4 000 т**, CAPEX 800 млн ₽ |
| Халяль-блок | да (цена бройлера 205 ₽/кг с учётом халяль/разделки) |
| Режим | 1–2 смены под план отгрузок |

## Поток

Убой → разделка → охлаждение → фасовка → холодильник → отгрузка.

Принцип «чистых» и «грязных» потоков, ветеринарные разрывы между зонами.

- SINT **2 400 голов/час кроликов** — **не применимо**
