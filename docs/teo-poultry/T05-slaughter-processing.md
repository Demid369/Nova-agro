## Убойно-перерабатывающий комплекс

| Параметр | Значение |
|----------|----------|
| CAPEX УПК | **2 600 млн ₽** |
| **Вендор (рекомендация)** | **Мелеузский МЗ + PKM-79** (РФ); альт. **Luohe Longfeng** (CN) |
| **Бройлерная линия** | **6 000 голов/ч** × 8 ч = 48 000/день (средняя нагрузка ≈52%) |
| Линия индейка | 500 гол/день |
| Утка / гусь | 2 000 гол/день |
| Холодильный склад | **4 000 т**, CAPEX 800 млн ₽ (**ААБ / Khladotech**) |
| **Халяль-блок** | да — **внутренний рынок РФ** (205 ₽/кг); **не экспорт** |
| Режим | 1–2 смены под план отгрузок |

## Поток

Приёмка → оглушение / **халяль-рез** → шпарка → оперение → потрошение → VetIS → охлаждение → разделка → фасовка → холодильник → отгрузка.

Перо, кровь, жир → переработка (Amandus Kahl или **Волжский завод**).
