# T11. Narrative и миссия

> Источник: `appendix/03-svodnyy-raschet-kompleksa.md`

## Резюме проекта

За **12 млрд ₽** создаётся **вертикально интегрированный птицеводческий комплекс полного цикла** — направление агрохолдинга **«Нова-Агро»** (**Запорожская область**).

**Ставка:** бройлер + куриное яйцо (≈70% CAPEX, >90% выручки) + диверсификация (индейка, утка, гусь, перепёл).

## Локация

| | |
|--|--|
| **Юридический адрес** | Запорожская область, г. Мелитополь, просп. Богдана Хмельницкого, д. 24А |
| **Производственная площадка** | **нет** — появится после выделения земли под комплекс |
| **Земельный участок (оценка из материалов)** | 150–400 га — *требование к будущей площадке, не текущий актив* |

## Юридическая структура

- **ООО «Нова-Агро — Птицеводство»** — отдельная бизнес-единица с собственным P&L
- Под управлением УК и функциональных директоров холдинга
