# T07. Побочная продукция и безотходность

> **Канон:** `energy-budget.yaml`, baseline кроликов (табл. #22, #205). Отличие — **помёт птицы** вместо навоза кролика.

## Органика (= кролики)

| Параметр | Кролики | Птица |
|----------|---------|-------|
| Сырьё | навоз кролика | **помёт птицы** |
| Линия компost | **5 т/ч** | **5 т/ч** |
| CAPEX компost | **36 млн ₽** | **36 млн ₽** |
| Удобрение | **43 800 т/год** | **43 800 т/год** |
| Выручка (12 ₽/кг) | **525,6 млн ₽/год** | **525,6 млн ₽/год** |
| Поток в биogaz APK | навоз → **20 МВт·ч** APK | помёт → **20 МВт·ч** APK |

## Прочие побочные потоки (только птица)

| Поток | Использование |
|-------|----------------|
| Перья / субпродукты убоя | мясоперьевая мука, технический жир |
| Яйцо (брак) | переработка / корм |

## vs кролики

| | Кролики | Птица |
|---|---------|-------|
| Шкурки / мех | **да** | **нет** |
| Связь с желатином/кожей | сильная | слабее |
