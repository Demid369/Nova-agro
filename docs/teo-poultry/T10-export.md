# T10. Экспорт

> **План проекта:** `appendix/export-plan-friendly-countries.md`  
> **База finmodel:** **0%** экспорта (консервативно). **Цель с 4-го года:** **6%** объёма мяса в **дружественные** страны.

## Статус

| Параметр | Значение |
|----------|----------|
| Целевые рынки | **ЕАЭС, СНГ, Иран, ОАЭ, Египет, Азербайджан**; резерв Вьетnam, Алжир |
| **Исключено** | Недружественные юрисдикции (ЕС, США, UK, JP, KR, UA и др.) |
| Объём (устойчивое состояние) | **1 090 т** мяса/год (**6%** от ≈18 170 т) |
| Выручка экспорт | **235 млн ₽/год** (≈4,2% общей выручки) |
| HS | **0207**, **020714**, **020727** |
| Халяль | **100%** экспортного потока |
| Firm-контракты | **нет** |
