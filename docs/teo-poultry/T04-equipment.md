# T04. Оборудование и инфраструктура

> Источники: `appendix/01-tehnologicheskiy-cikl.md`, лист «CAPEX» в finmodel

## Технологические зоны

| Зона | Состав | Назначение |
|------|--------|------------|
| Воспроизводство | Инкубаторий 500 тыс. яиц/нед. | Суточный молодняк |
| Выращивание | **118 птичников/модулей** | Откорм по видам |
| Убой и переработка | УПК, цех упаковки, **халяль-блок** | Разделка, фасовка |
| Хранение | Холодильный склад **4 000 т** (−18 / +2 °C) | Логистика |
| Корма | Силоса, приёмка | ~66,7 тыс. т/год с ККЗ холдинга |
| Энергетика (канон блока) | Солнечные панели **10 МВт·ч** на крышах | как baseline кроликов |
| Энергетика (finmodel, справ.) | Биогаз + ГПУ на площадке | CAPEX **450 млн ₽** — пересмотреть при sync finmodel |
| Инженерия | ЛЭП, подстанция, водоснабжение | CAPEX **1 300 млн ₽** |

## CAPEX по группам (из finmodel)

- Бройлер + яйцо (птичники, сортировка яйца): **2 405 млн ₽**
- Прочие виды (индейка, утка, гусь, перепёл): **1 025 млн ₽**
- Общая инфраструктура (убой, холод, инкубаторий, сети…): **8 570 млн ₽**

## Замена vs кролики

- **Meneghin Srl** — не используется
- Собственные линии птицеводства + **УПК 2 600 млн ₽**
