# Сценарий: птицеводство вместо кролиководства

> **Статус:** **parked** — второе ТЭО, не рабочий проект. Эвристика от baseline, не DCF.

## Суть замены

| | Базовый (кролики) | Вариант (птица) |
|---|-------------------|-----------------|
| Продукт | 7 000 т крольчатины/год | ~12 000 т мяса птицы/год (оценка) |
| CAPEX | 12 млрд | ~17 млрд |
| NPV (тыс.) | 2 779 519 | 4 050 156 (эвристика) |
| IRR | 15,19% | 18,49% |
| Оборудование | Meneghin Srl | Бройлерные/утиные линии (TBD) |
| Экспорт | Крольчатина (нишевый) | HS 0207 — птица (ОАЭ +220% в 2024) |

## Что сохраняется из инфраструктуры

1. **Комбикормовый завод** — в `03-proizvodstvo` уже есть рецепты и премиксы для птиц
2. **Убойный цех** — нужна линия птицеубоя (SINT или аналог)
3. **Биогаз** — птичий помёт, другая агрохимия vs кроличий навоз
4. **Экспортный кластер** — СНГ, ОАЭ; HS 0207 хорошо описан в `04-rynok`

## Что теряется / ослабевает

- **Meneghin Srl**, 81 цех кролиководства, ANCI
- **Мех и шкурка кролика** → заменяется перьевой/мясоперьевой мукой (меньше маржа)
- **Модель нулевой себестоимости мяса** — с птицы мало сырья для желатина/кожи
- **Диетический нишевый продукт** — кролик vs массовый бройлер
